Connecting To Dog

How to Connect to Dog Wired

  1. Have the dog on the ground flat, turned on.
  2. Plug in an hdmi cable and a keyboard and/or mouse into the right side on the Robot Dog's back.
  3. Open up a terminal and type "sudo ifconfig wlan0 up" and "sudo ifconfig wlan2 up" to turn on the dog's internet for computers to wirelessly connect to the dog.

Connect Wirelessly

  1. Use "PuTTy" to remote access one of the dogs.
  2. Type in the RoboDog's ip in the Host Name box.
  3. Login as pi, password is 123.

Send File To Dog

  1. Type in "pscp filename pi@ip/home/pi"
